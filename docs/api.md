# API

Versioned API prefix: `/api/v1`.

Implemented local demo endpoints: `/me`, `/risk/evaluate`, `/demo/copy-trade`, `/demo/submit-order`, `/demo/twap`, `/demo/volume-execution`, `/demo/unknown-order`, `/kill-switch/account/{account_id}`, `/reconciliation/demo`, plus `/health`, `/ready`, and `/metrics`.

OpenAPI is generated by FastAPI at `/docs` and `/openapi.json`.
