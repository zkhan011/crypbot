Current tests cover Decimal rounding, risk rejection, kill switches, copy state transitions, TWAP slicing, compliant volume-aware participation caps, mock live telemetry fields, credential encryption/redaction, BingX signing fixtures, idempotent unknown-order recovery, and reconciliation mismatches. The roadmap adds PostgreSQL repository tests, Redis lock tests, authorization/tenant isolation API tests, WebSocket duplicate-event tests, and browser E2E dashboard workflows.
